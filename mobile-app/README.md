# Mobile App (React Native)

Aplicação mobile para gerenciamento de usuários consumindo uma API REST.

---

## Tecnologias

- React Native
- Expo
- TypeScript

---

## Funcionalidades

- Login e autenticação
- Listagem de usuários
- Edição de usuários
- Exclusão de usuários
- Paginação
- Busca

---

## Como rodar

```bash
npm install
npx expo start
```

---

## Observações

- Consome API construída com NestJS
- Gerenciamento de estado com hooks (useState, useEffect)
- Interface pensada para mobile com feedback visual (loading, alerts)
